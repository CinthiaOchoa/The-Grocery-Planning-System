# 🥗 Grocery Planning System

## 📌 Overview
The Grocery Planning System is a full-stack web application that helps students manage their pantry, track groceries, and plan meals based on time and budget.

Users can store ingredients, add recipes, track expiration dates, and receive meal recommendations.

---

## 🚀 Features
- Add and manage recipes  
- Track pantry items and expiration dates  
- Record grocery purchases  
- Update and delete pantry items  
- View expiring ingredients  
- Get recipe recommendations based on time and budget  

---

## 🛠️ Technologies
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express  
- Database: MySQL
